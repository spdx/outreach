# SPDX Absolute Beginner Guide

## What is SPDX?
SPDX (Software Package Data Exchange) is a standard format used to describe the components, licenses, and metadata of software.

## Why is it important?
It helps organizations:
- Understand what software they are using
- Ensure license compliance
- Improve security and transparency

## Prerequisites
- Basic understanding of software projects
- Internet browser

## Step-by-Step Guide

### Step 1: Open SPDX Online Tool
Go to: https://tools.spdx.org/app/

### Step 2: Enter Project Information
Add:
- Project name (e.g., SampleApp)
- Version (e.g., 1.0.0)

### Step 3: Add Packages
List the software components used in your project.

### Step 4: Add License Information
Specify licenses for each component if known.

### Step 5: Export Document
Download the SPDX file in your preferred format (JSON, RDF, etc.)

## Example Output
Your SPDX document will include:
- Project details
- Package list
- License data
